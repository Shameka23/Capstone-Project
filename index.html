Variables (click to expand)

Level: Beginner

Variables store data that can be used later in a program.

Example:

x = 10
name = "Shameka"
print(x)
print(name)

Loops

Dictionaries

    "grade": "A"
}

    print(10 / 0)
except ZeroDivisionError:
    print("Cannot divide by zero")
