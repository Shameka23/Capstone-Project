What is Python?

Python is a beginner-friendly programming language used for building applications, data analysis, automation, and more.

Variables

Variables store data values in a program.

x = 10
name = "Shameka"
print(x)

Lists

Lists store multiple values in a single variable.

numbers = [5, 10, 15]
print(numbers)

Conditional Statements

Conditionals allow programs to make decisions.

age = 18
if age >= 18:
    print("Adult")
else:
    print("Minor")

Functions

Functions allow code to be reused.

def greet(name):
    return "Hello " + name

print(greet("Shameka"))
